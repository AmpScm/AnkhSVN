AnkhSVN now on openCollabNet

AnkhSVN is a Visual Studio plugin for the Subversion, it allows you to perform the most common version control operations directly from inside the Microsoft Visual Studio IDE. With AnkhSVN you no longer need to leave your IDE to perform tasks like viewing the status of your source code, update your Subversion working copy and committing changes. You can even browse you repository and you can plu-in you favorite diff tool.

AnkhSVN is an active open source project with multiple committers from around the world (including CollabNet). AnkhSVN is currently in release 1.02 and we are working hard on the next release. If you like AnkhSVN and use it daily, join our development team (we have quite a few features on our roadmap).
