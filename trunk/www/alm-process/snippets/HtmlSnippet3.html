Optimized workflow

  • No need to leave the IDE for most common operations.
  • Immediately view the source control status of all files in your project/solution.
  • Visual Studio properties window extended to show working copy information such as last committed author, last committed date, repository URL etc for files and folders.
  • Streamlined import of new solutions directly from the solution explorer.
  • Supports all Subversion transfer protocols.
Pluggable diff/merge
  • Plug in your diff/merge tool of choice.
  • Command line templates for the most common merge tools.
Working Copy Explorer
  • Easily work with files that aren't part of your projects/solution.
  • Use the same commands as in the Visual Studio solution explorer.
Repository Explorer
  • Easily browse any Subversion repository
  • View extended information about remote files and directories in the Visual Studio Properties window
