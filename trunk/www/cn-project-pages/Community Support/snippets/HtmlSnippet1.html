Before filing a new issue, please:

  • Read the Frequently Asked Questions.
  • Search the AnkhSVN issue database and check if the issue is already known.
  • Verify if the issue is still an issue in the latest daily build.
  • If you are not absolutely sure the issue is actually a bug, ask in the support forum.
  • Read the documentation on how to submit an issue.
  • Submit your defect report .
Please follow this process and keep the bug database clean so the AnkhSVN committers can focus on developing new code, rather than vetting the bug database.
