AnkhSVN is an addin that marries Visual Studio with the Subversion version control system, which was named sole leader in stand-alone SCM by Forrester Research. Subversion's leadership position comes from its ease of use, out-of-the-box low-cost support for distributed teams, and enterprise-class scalability. Visual Studio is the dominant development environment for .NET and native Windows development.
